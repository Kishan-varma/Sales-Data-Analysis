# Sales Data Analysis Project

## Overview
This project analyzes a retail sales dataset to uncover trends, patterns, and actionable business insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Analysis Performed
- Data cleaning and preprocessing
- Feature creation (Month, Year, Shipping Days)
- Sales distribution analysis
- Region-wise and category-wise performance analysis
- Customer and shipping mode analysis

## Key Insights
- West region generates the highest revenue
- Technology category performs best
- Standard shipping mode contributes the most sales
- Top customers significantly impact overall revenue
